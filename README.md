# Customer Shopping Behavior Analysis

## Project Overview
This project focuses on analyzing customer shopping behavior using Python, SQL, MySQL, and Power BI. The goal of the project is to identify customer purchasing patterns, spending trends, product performance, and business insights through data analysis and visualization.

## Tools Used
- Python
- Pandas
- MySQL
- SQL
- Power BI

## Key Highlights
- Cleaned and processed customer shopping dataset
- Performed SQL analysis for revenue and customer insights
- Built interactive Power BI dashboard for visualization
- Analyzed spending trends and category performance
